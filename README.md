# Building backend

Pre-requisites:
 - Node.js
 - NPM

Run these commands to build and run the backend:
```
git clone https://github.com/The-Mobile-Workshop/PhoneDB.git
cd PhoneDB/backend
npm i
npm run build
cd build
node .
```
